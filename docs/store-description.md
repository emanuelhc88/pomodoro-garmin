## Long description (EN)

```
Toma Pomodoro is a focus timer built for Garmin wearables. Run work/break
cycles straight from your wrist — no phone needed.

FEATURES
• Three built-in presets: 25/5, 30/5, 50/10 (work/break in minutes)
• Custom preset builder: 5–90 min work, 1–30 min break, 1–10 cycles
• Work → Break → Long Break state machine with visual progress ring
• Vibration, sound and backlight alerts on phase changes (respects DND)
• Optional FIT activity recording synced to Garmin Connect
• Local session history and daily session counter
• Session recovery if the app is closed mid-session
• Portuguese and English (auto-detected, override in Settings)

PERMISSIONS
• Fit — used only to create the optional FIT activity file when "Record as
  activity" is enabled in Settings.

PRIVACY
No data is collected, transmitted, or shared. Everything stays on your device.

BATTERY
The timer keeps the screen active during a running session, which increases
battery consumption for the duration of the session.

SUPPORTED DEVICES
Forerunner 255, 255S, 255 Music, 265, 265S, 955, 965
Fenix 7, 7 Pro, 8 (43mm), 8 (47mm)
Epix Gen 2, Venu 3, Venu 3S, Vivoactive 5

SUPPORT
https://github.com/emanuelhc88/pomodoro-garmin/issues
```

## Long description (PT)

```
Toma Pomodoro é um timer de foco feito para wearables Garmin. Rode ciclos de
trabalho e pausa direto do pulso — sem precisar do celular.

RECURSOS
• Três presets prontos: 25/5, 30/5, 50/10 (foco/pausa em minutos)
• Preset personalizado: 5–90 min de foco, 1–30 min de pausa, 1–10 ciclos
• Máquina de estados Foco → Pausa → Pausa Longa com anel de progresso
• Alertas de vibração, som e backlight nas trocas de fase (respeita DND)
• Gravação opcional como atividade FIT sincronizada com o Garmin Connect
• Histórico local de sessões e contador diário
• Recuperação de sessão se o app for fechado durante a execução
• Português e inglês (detecção automática, ajustável em Configurações)

PERMISSÕES
• Fit — usada apenas para criar o arquivo FIT quando "Gravar como atividade"
  estiver ativa.

PRIVACIDADE
Nenhum dado é coletado, transmitido ou compartilhado. Tudo fica no seu device.

BATERIA
O timer mantém a tela ativa durante a sessão, o que aumenta o consumo de
bateria pelo tempo da sessão.

DISPOSITIVOS SUPORTADOS
Forerunner 255, 255S, 255 Music, 265, 265S, 955, 965
Fenix 7, 7 Pro, 8 (43mm), 8 (47mm)
Epix Gen 2, Venu 3, Venu 3S, Vivoactive 5

SUPORTE
https://github.com/emanuelhc88/pomodoro-garmin/issues
```
